# 医院药房智能管理系统 - 任务清单

## 📋 项目概述
医院药房智能管理系统是一个基于Java Spring Boot + Python Flask + Vue3的全栈应用，包含药品管理、库存管理、处方审核等核心功能。

**开发时间**: 2024年12月9日 - 2025年1月31日
**项目周期**: 8周

## 👥 团队分工

### 李教博 (Java后端) - 70%工作量
负责Spring Boot后端API开发、数据库设计、用户认证与权限管理

### 田纹搴 (Python审核服务) - 20%工作量
负责药品审核算法、审核服务API、数据分析功能

### 易瑜 (前端Vue3) - 10%工作量
负责用户界面开发、API集成、用户体验优化
